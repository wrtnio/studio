Type Alias IStudioWorkflowCompileResult

IStudioWorkflowCompileResult: IStudioWorkflowCompileResult.ISuccess | IStudioWorkflowCompileResult.IFailure

스튜디오 컴파일 결과물.

IStudioWorkflowCompileResult 는 스튜디오의 IStudioWorkflow 워크플로우 개별 문서나, IStudioRepositoryRelease 저장소 배포본을 빌드/컴파일한 결과를 형상화한 엔티티이다.

속성 success 값을 통해 성공 여부를 판단하며, 이를 기준으로 유니언 타입이 특정되는 discriminated union 구조이다. 성공 시에는 IStudioWorkflowCompileResult.ISuccess 타입이, 실패 시에는 IStudioWorkflowCompileResult.IFailure 타입이 특정된다.
