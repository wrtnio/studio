IStudioWorkflowConnector: IStudioRepositoryReleaseNode | IStudioWorkflowScriptConnector | IStudioWorkflowSwaggerConnector

워크플로우 문서상 배치된 커넥터 함수의 정보.

IStudioWorkflowConnector 는 유니언 타입의 일종으로써, IStudioWorkflow 워크플로우에 배치한 함수 커넥터를 형상화한 엔티티이다. 그리고 여기서 말하는 함수란, 워크플로우 문서에서 IStudioWorkflowCallStatement 노드로 호출할 수 있는 아래 서브타입의 것들을 뜻한다.

  • IStudioRepositoryReleaseNode: 릴리즈로부터 불러온 함수
  • IStudioWorkflowScriptConnector: 내부 저장소로부터 불러온 함수
  • IStudioWorkflowSwaggerConnector: 외부 Rest API 를 호출하는 함수
