Interface IStudioMetaChatService

Meta LLM 서버가 클라이언트에게 제공하는 함수 셋.

IStudioMetaChatService 는 Meta LLM 클라이언트에게, 서버가 제공하는 함수 셋을 정의한 인터페이스이다. Meta LLM 클라이언트는 이 인터페이스를 통해, 서버의 IStudioMetaChatService 함수들을 RPC (Remote Procedure Call) 한다.

참고로 클라이언트는 Meta LLM 서버에 접속한 후, 곧바로 initialize 함수를 호출해줘야 한다. 만일 initialize 함수 호출을 건너뛰고 바로 talk 부터 시도하려 든다면, 해당 함수 호출은 실패한다.

  • 채팅 세션 발행/복원/초기화.

    클라이언트가 Meta LLM 서버에 접속한 후, 최초 1 회 호출하여 채팅 세션을 발행하거나 또는 기존 세션을 복원해줘야 한다. 만일 initialize 함수 호출을 건너뛰고 바로 talk 부터 시도하려 든다면, 해당 함수 호출은 실패한다.

    그리고 리턴되는 세션은 신규 채팅 세션 발행 API 로 접속하였다면 새 세션을 발행하여 리턴해주고, 기존 세션에 이어나가는 API 로 접속하였던 경우에는 기존 세션을 복원하여 리턴해준다.
