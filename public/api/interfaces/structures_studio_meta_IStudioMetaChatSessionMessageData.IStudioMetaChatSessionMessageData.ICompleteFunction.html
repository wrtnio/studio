reaction

reaction: IExecutionResult<any>

대상 RPC 함수의 리턴값.

서버에서 클라이언트로의 RPC 함수 호출 시에는 클라이언트 함수의 리턴값, 그리고 클라이언트에서 서버로의 RPC 함수 호출 시에는 서버 함수의 리턴값이다.

단, 함수 호출이 언제나 성공하는 것은 아니기에, 리턴값이 아닌 에러가 발생하였을 수 있기에, returned value 및 thrown exception 의 유니언 타입에 해당하는 IExecutionResult 가 쓰였다.

본래 속성명이 return 인게 의미상 맞으나, 예약어라 일부러 피했다.
