Interface IHubSaleUnitStockPrice

스톡의 가격 정보.

IHubSaleUnitStockPrice 는 스톡의 계단식 가격 정보를 형상화한 엔티티이다. 그리고 여기서 말하는 계단식 가격이란, 각각이 고정비와 변동비를 모두 아우르며, 그러한 구간이 여럿 있는 것을 뜻한다.

계단형 가격 정보가 무엇인지는, 아래 예시를 보면 쉬이 이해할 수 있을 것이다.

  • 고정비 50,000 원에 API 1,000 건까지 무료, 초과분 1건당 100 원
  • 고정비 100,000 원에 API 2,500 건까지 무료, 초과분 1건당 75 원
  • 고정비 150,000 원에 API 4,000 건까지 무료, 초과분 1건당 50 원
