Interface IHubCouponCriteriaOfFunnel

할인 쿠폰의 유입 경로 제한.

IHubCouponCriteriaOfFunnel 은 IHubCouponCriteria 의 서브타입 엔티티로써, IHubCoupon 할인 쿠폰을 특정한 경로에서 유입된 고객에게만 발급 또는 배제하고자 할 때 사용한다.

그리고 유입 경로 제한은 다음과 같이 3 가지 방식으로 가능하다. 첫째는 IHubCustomer.referrer 이며, 이외에 고객의 접속 주소를 기록한 IHubCustomer.href 를 파싱함으로써, 특정 URL 내지 변수 단위로 제약이 가능하다.
