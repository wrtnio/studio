Interface IStudioWorkflowSwaggerConnector

워크플로우상 스웨거 함수의 커넥터 정보.

IStudioWorkflowSwaggerConnector 는 IStudioWorkflow 워크플로우에 배치한 스웨거 함수의 커넥터를 형상화한 엔티티이다. 그리고 여기서 말하는 "스웨거 함수" 란, 워크플로우 문서에서 IStudioWorkflowCallExpression 노드로 호출할 수 있는 외부 Rest API 와 그것의 메타데이터 스펙을 뜻한다.

만일 호출하고자 하는 함수가 외부 Rest API 서버의 것이 아닌 동 IStudioRepository 저장소에 타입스크립트로 작성된 것이라면, IStudioWorkflowScriptConnector 타입을 대신 사용하도록 한다. 그리고 만일 호출하고자 하는 함수가 타 저장소의 IStudioRepositoryRelease 릴리즈 본에서 배출 (export) 된 것이라면, IStudioWorkflowReleasedConnector 타입을 대신 사용한다.
