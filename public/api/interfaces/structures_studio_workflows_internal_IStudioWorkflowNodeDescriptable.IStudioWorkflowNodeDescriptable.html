Interface IStudioWorkflowNodeDescriptable

설명 가능 노드.

IStudioWorkflowNodeDescriptable 는 IStudioWorkflowStatement 워크플로우 노드문 중, 변수를 생성하며 그 변수에 대하여 따로이 title 및 description 정의가 필요한 노드문들을 형상화한 슈퍼타입 엔티티이다.

가령 for each 문을 형상화한 IStudioWorkflowForEachStatement 의 경우 배열 변수를 하나 생성해내는데, 이 때 해당 배열 변수가 IStudioWorkflowSetReturnValueStatement 에 의하여 리턴되는 경우, 해당 변수에 대하여는 설명이 누락된다. 그리고 이 설명이 하필이면 LLM 의 퀄리티를 좌우할만큼 중요한 요소이다.

또한 IStudioWorkflowCallStatement 는 외부 커넥터 함수를 참조하기에, title 및 description 을 해당 함수들로부터 취득할 수도 있다. 하지만 사용자가 커넥터 함수와 별개로, 해당 노드로부터 생성되는 변수에 별도로 설명을 적고싶을 수도 있다. 이 때, title 과 description 를 채워주면 된다.

따라서 본 타입의 속성 title 및 description 을 작성한 경우, IStudioWorkflowData.output 워크플로우의 리턴 타입을 형상화한 ISwaggerSchema에, 해당 배열 변수에 대한 상세 설명이 반영된다.
