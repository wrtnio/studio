Interface IHubDepositDonation

예치금 공여. IHubDepositDonation 는 관리자가 특정 고객 시민에게 예치금을 직접 공여하는 경우를 형상화한 엔티티이다.

고객이 예치금을 우리의 결제 시스템을 통하지 않고, 대면으로 직접 전달해주는 경우를 상정하고 설계하였다.

경우에 따라서는 특정 기업과 후불 요금제 계약을 맺고, 그 기업에게 매달 적당한 수준의 예치금을 부여하는 경 우 또한 생각해 볼 수 있다.
