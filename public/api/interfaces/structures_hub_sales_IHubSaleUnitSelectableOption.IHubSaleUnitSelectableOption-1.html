Interface IHubSaleUnitSelectableOption

매물 유닛에서의 개별 선택형 옵션 정보.

IHubSaleUnitSelectableOption 은 매물 내 개별 제품을 형상화한 IHubSaleUnit 의 하위 엔티티로써, 해당 유닛의 개별 선택형 옵션 정보를 형상화하기 위하여 설계된 엔티티이다.

  • 선택형 옵션의 사례
    • 컴퓨터 성능: CPU, GPU, RAM 등
    • 이미지 생성 AI: 랜더링 퀄리티, 라이선스 종류, 테마 등

그리고 만일 본 엔티티의 속성값 IHubSaleUnitSelectableOption.variable 이 true 라면, 이는 고객이 본 옵션에서 어떠한 후보값을 선택하냐에 따라, 구매하게 될 IHubSaleUnitStock 최종 구성품이 달라지는 것을 뜻한다.

반대로 위 값이 false 라면, 고객이 본 옵션에서 어떠한 후보값을 선택하던, 고객이 구매하게 될 IHubSaleUnitStock 최종 구성품에는 영향을 주지 않는다. 판매자가 참고용으로만 사용하게 될 단순 정보에 불과할 뿐.
