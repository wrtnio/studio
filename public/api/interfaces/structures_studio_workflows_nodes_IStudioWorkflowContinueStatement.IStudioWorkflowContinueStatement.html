Interface IStudioWorkflowContinueStatement

워크플로우 상 반복문을 즉시 재시작하는 Continue 노드.

IStudioWorkflowContinueStatement 는 IStudioWorkflow 워크플로우 프로그램 상 IStudioWorkflowForEachStatement 나 IStudioWorkflowWhilteStatement 등의 반복문 내에서 반복문을 즉시 재시작하는 Continue 문을 형상화한 노드 엔티티로써, 오직 위 엔티티들의 반복문 내에서만 사용 가능하다.

그리고 본디 반복문 내에서 IStudioWorkflowCallStatement 커넥터 함수 호출 등으로 지역 변수가 생겨난 경우, 이들은 모두 반복문 블록 시작 전에 선언된 배열에 삽입된다. 그러나, 반복문 내에서 본 IStudioWorkflowContinueStatement 나 IStudioWorkflowBreakStatement break 문 등이 쓰인 경우라면, 지역 변수들은 상위 배열에 삽입되지 않는다.

const array = [];
for (const elem of array) {
   ...(some-statements)
   continue;
   array.push({
     ...variables generated in the block
   });
}
