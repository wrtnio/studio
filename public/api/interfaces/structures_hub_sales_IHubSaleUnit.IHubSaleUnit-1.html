Interface IHubSaleUnit

매물의 개별 유닛 정보.

IHubSaleUnit 은 매물에서 취급하는 "개별 제품" 정보를 형상화한 엔티티이다. 그리고 여기서 말하는 "개별 제품" 이란, 허브 시스템에서의 개별 서버 단위, 즉 스웨거 문서에 대응하는 개념이다.

참고로 IHubSaleUnit 이 IHubSaleSnapshot 으로부터 1: N 의 대수관계로 분리된 이유는 바로, 하나의 매물에서 복수의 제품을 판매하는 경우가 종종 있기 때문이다. 이른바 "묶음 상품" 의 경우가 그러하다.

  • 일반 상품에서의 묶음 상품 (노트북 세트)
    • 본체
    • 키보드
    • 마우스
    • 애플 케어 (무료 A/S 권)
  • 허브 시스템에서의 묶음 상품 (작가 AI 세트)
    • 글쓰기 서버
    • 이미지 생성 서버
    • 로고/표지 생성 서버

그리고 IHubSaleUnit 은 다시, 그 자체로써 IHubCustomer 고객이 구매하게 될 최종 구성품을 의미하지는 않는다. 최종 구성품은 주어진 옵션과 그 IHubSaleUnitOptionCandidate 후보값들을 모두 선택한 끝에라야, 비로소 만나볼 수 있는 것.

가령 노트북을 한 대 사더라도, 그 안에서 각각 (CPU / RAM / SSD) 등의 옵션을 모두 고른 뒤에야, 최종 구성품이 정해지듯이 말이다.
