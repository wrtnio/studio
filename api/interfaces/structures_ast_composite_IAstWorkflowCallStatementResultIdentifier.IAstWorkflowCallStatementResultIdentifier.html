Interface IAstWorkflowCallStatementResultIdentifier

워크플로우 커넥터 함수 호출문의 리턴 결과 표현식.

IStudioWorkflow 워크플로우 문서에서 커넥터 함수를 호출한 경우, 이 때의 함수 호출문은 IStudioWorkflowCallStatement 로 표현된다. 그리고 커넥터 함수에서의 리턴값은, 모종의 변수에 저장된다.

IAstWorkflowCallStatementResultIdentifier 는 바로 이러한 커넥터 함수의 리턴값을 저장한 모종의 변수를 식별하는 표현식이다. 즉 IAstWorkflowCallStatementResultIdentifier 는 IAstIdentifier 의 일종이되, 커넥터 함수 호출문의 리턴값을 저장한 모종의 변수를 위한 전용 식별자 표현식인 것.
