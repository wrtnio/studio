Interface IHubSection

섹션 정보.

IHubSection 은 시장에서의 섹션 정보를 표현하기 위하여 설계된 엔티티이다.

여기서 말하는 섹션을 마트에 비유하면, "과일코너" 나 "정육코너" 등 매장 내에서 공간적으로 분리된 영역을 뜻한다. 현재 뤼튼의 Generative Hub 에 존재하는 섹션은 오직 "API 마켓" 하나 뿐이나, 앞으로의 확장성을 위해 미리 설계해 둔 개념이다.

그리고 하나의 IHubSale 매물에서 동시 분류가 가능한 IHubChannelCategory 와는 달리, 섹션은 하나의 매물에 대해서만 분류가 가능하다.
