Interface IAstArrayMapExpression

Array.map() 함수 표현식.

IAstArrayMapExpression 은 원본 배열을 또다른 타입의 배열로 변환해주는 Array.map() 함수를 형상화한 인터페이스이다.

각각 expression 과 transformer 는 대상 배열 및 변환식을 나타내며, id 는 Array.map() 이 사용하게 될 콜백 함수의 개별 (파라미터) 원소 및 인덱스 번호를 식별하는데 사용되는 데 쓰인다. 구체적으로는 콜백 함수의 파라미터 Array.map((element, index) => ...) 에서 개별 원소를 지칭하는 IAstArrayFunctionParameterIdentifier.expression_id 및 인덱스 번호를 형상화한 IAstArrayFunctionIndexIdentifier.expression_id 를 위하여 존재한다.

아래는 IAstArrayMapExpression 의 활용 예시.

코드 예시

const array: number[] = [...];
array.map((value, index) => {
  return {
    id: value + index,
    age: 20,
  };
});
