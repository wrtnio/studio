Interface IStudioMetaChatFunctionCallPlan

Meta Chat Agent 의 함수 호출 계획 설명.

IStudioMetaChatFunctionCallPlan 은 Meta LLM 채팅 세션에서, 서버가 클라이언트에게 함수 호출 계획을 설명할 때 사용하는 인터페이스이다. Meta LLM 이 어떠한 함수들을 호출할 것이고, 그 호출 계획에 대한 전체/개발 수준의 설명을 담고 있다.

RPC (Remote Procedure Call) 상, IStudioMetaChatListener.explainFunctionCall 함수에서 사용된다.
