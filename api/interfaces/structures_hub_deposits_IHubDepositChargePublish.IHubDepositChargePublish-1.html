Interface IHubDepositChargePublish

예치금의 입금에 대한 결제 진행 정보.

IHubDepositChargePublish 은 고객이 예치금 입금을 신청하고, 이에 관하여 결제를 진행하는 과정을 형상화한 엔티티이다.

참고로 IHubDepositChargePublish 레코드의 존재가 곧 결제의 완료를 뜻하는 것은 아니다. 결제는 지불이 끝나야 비로소 완성되는 것이다.

위에서 말한 "결제를 진행하는 과정" 이란 바로 이러한 의미인 것.

다만 지불이 이루어진 이후로도, 돌연 그것이 취소되는 경우가 있으니, 이 또한 주의해야 한다.
