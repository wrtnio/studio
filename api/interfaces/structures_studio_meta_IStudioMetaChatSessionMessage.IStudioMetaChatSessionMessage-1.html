Interface IStudioMetaChatSessionMessage<Data>

Meta LLM 채팅 세션 메시지 정보.

IStudioMetaChatSessionMessage 는 IStudioMetaChatSession Meta LLM 채팅 세션에서 발생한 메시지를 기록한 엔티티이다. 정확하게는 채팅 세션 내에서도 어느 IStudioMetaChatSessionConnection WebSocket connection 에서 발생한 것인지를 기록하고 있다.

그리고 우리의 Meta LLM 채팅 세션은 웹소켓 프로토콜에 TGrid 의 RPC (Remote Procedure Call) 개념을 사용하고 있기에, 본 IStudioMetaChatSessionMessage 는 곧 RPC 함수 호출에 대한 이력 정보 IStudioMetaChatSessionMessageData 를 포괄하는 엔티티이기도 하다.
