Interface IStudioRepositoryCommit

저장소 커밋 정보.

IStudioRepository 저장소를 구성하는 각각의 버킷들을 편집하고, 그 이력을 기록하는 행위를 "커밋" 이라고 한다. 그리고 IStudioRepositoryCommit 는 저장소에 쓰기 권한을 가진 고객이 행한 커밋 행위를 형상화한 엔티티이다.

즉 본 커밋 레코드가 하나 생성되었다는 것은, 최소 한 개 이상의 버킷이 수정되었고, 그리하여 그들에 대한 스냅샷 레코드들이 새로이 생성되었다는 것을 뜻한다.

단 커밋을 발행하였다하여, 그것이 곧 커밋의 완결을 뜻하는 것은 아니다. 커밋은 그것의 기록 확정 일시를 뜻하는 IStudioRepositoryCommit.archived_at 속성이 null 인 경우에는 버킷들을 계속 수정할 수 있으며, 심지어 현재 수정 중인 커밋을 취소하고 삭제할 수도 있다.
