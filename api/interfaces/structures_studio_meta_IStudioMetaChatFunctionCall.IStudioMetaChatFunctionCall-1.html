Interface IStudioMetaChatFunctionCall

Meta LLM 의 Chat Session 상 함수 호출 정보.

IStudioMetaChatFunctionCall 은 Meta LLM 채팅 세션 IStudioMetaChatSession 상에서 행한 함수 호출 내역 및 그에 대한 결과값 등을 형상화한 인터페이스이다.

참고로 여기서 말하는 함수 호출이란, LLM (Large Language Model) Function Call 을 뜻한다. 또한 LLM Function Call 이란, OpenAI 와 같은 LLM 이 사용자와의 대화 문맥으로부터 적절한 함수를 선택하고, 그 함수를 호출하는 데 필요한 인자값들을 구성해주며, 그 인자값들을 토대로 LLM Meta 채팅 서버 프로그램이 이를 실제로 실행하는 (execution) 일련의 과정을 뜻한다.

본 IStudioMetaChatFunctionCall 에서 사용되는 함수의 타입 정보 function 에 대해서는, IOpenAiFunction 및 @wrtnio/openai-function-schema 저장소를 참고하도록 하자.
