Type Alias IHubSaleUnitOption

IHubSaleUnitOption: IHubSaleUnitSelectableOption | IHubSaleUnitDescriptiveOption

매물 유닛에서의 개별 옵션 정보.

IHubSaleUnitOption 은 매물 내 개별 제품을 형상화한 IHubSaleUnit 의 하위 엔티티로써, 해당 유닛의 개별 옵션 정보를 형상화하기 위하여 설계된 엔티티이다.

그리고 각각 선택형 옵션과 서술형 옵션을 뜻하는 IHubSaleUnitSelectableOption 및 IHubSaleUnitDescriptiveOption 의 유니언 타입 엔티티로써, 각 타입에 대한 특정화는 아래와 같이 type 속성에 대한 if condition 으로 간단히 이뤄낼 수 있다.

if (option.type === "select")
    option.candidates; // IHubSaleUnitSelectableOption

만일 옵션의 타입이 select 에 가변값이 참이라면, 그 후보값의 선택에 따라, 고객이 구매하게 될 IHubSaleUnitStock 최종 구성품이 달라진다.

반대로 select 이외에 타입이라던가, 타입은 select 이되, 가변 여부가 거짓이라면, 이는 단순 정보 전달 이상의 의미가 없는 옵션이다. 따라서 고객이 이를 구매하며 그 어떠한 값을 기재하고 선택하던, 이 경우의 옵션은 IHubSaleUnitStock 최종 구성품에 영향을 주지 않는다.
